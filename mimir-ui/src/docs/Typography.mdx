# Typography

**Primary Font:** Lato (self-hosted)  
**Defined Weights:** 100 (Thin), 300 (Light), 400 (Regular), 700 (Bold), 900 (Black)

The font family is declared as a semantic CSS variable (`--font-base`) in `theme.css`. Components should reference that variable instead of hard-coding a stack.
